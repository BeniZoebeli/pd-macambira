Random Pd Externals

This is a collection of random binaries I found on the net. There is no guarantee that they'll work for you. Consider this stuff very alpha. You will inevitably need to install some other stuff to get these working. For example, both GEM and PDP need a number of external libraries. The best way to get these is to use http://fink.sf.net .

These are the included binary packages:

  • IEMlib 1.12
  • GEM (G4) CVS 2003.08.06
  • PDP 0.12.1
  • pool 0.1.0
  • vasp 0.1.2
